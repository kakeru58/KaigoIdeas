介護福祉のアイデアサイト

このサイトは、介護福祉に関するアイデアや情報を共有するためのプラットフォームです。

使い方

以下のリンクから実際のアプリケーションを利用できます：

介護福祉のアイデアサイト

1. 新しい投稿を追加

  • タイトル: 投稿のタイトルを入力します。
  • 画像: 画像をアップロードします（オプション）。
  • 詳細: 投稿の詳細を入力します。
  • タグ: タグを追加できます。タグは自動補完機能を持ち、既存のタグから選択できます。

2. 投稿を検索

  • 検索キーワード: 投稿のタイトル、詳細、タグをキーワードで検索できます。
  • タグ検索: タグを選択して検索できます。タグは自動補完機能を持ち、既存のタグから選択できます。

3. 投稿の表示

  • 一覧表示: すべての投稿を一覧で表示します。各投稿にはタイトル、詳細、タグ、いいね数が表示されます。
  • 画像表示: 画像がアップロードされている場合、投稿の横に画像が表示されます。
  • タグ表示: タグは丸く囲んで表示されます。

4. いいね機能

  • 各投稿には「いいね」ボタンがあり、ユーザーは一度だけいいねを押すことができます。いいねを押すと、その投稿のいいね数が増えます。
